 ALT A — "why I built it" (builder narrative)
--------------------------------------------------------------------

Got tired of standing up 6 containers (Postgres + ClickHouse + Redis + MinIO…) just to self-host Langfuse and trace my coding agents.

So I built AgentTap: the same traces / sessions / cost views in one Node process + one SQLite file.

npx agenttap
